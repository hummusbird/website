Why use clapped?

IT'S FREE

Clapped is a bog-standard discord modbot. Nothing interesting, except it's COMPLETELY FREE. No premium features, no cooldown on commands, no shitty adverts, and no moaning for donations. (except on this website)

EASY SETUP

Clapped runs off a simple command based config. No dogshit web-interfaces that have 2% uptime. Add to your server, and it already works. Want to customise the prefix? the "FUCK YOU" reply? go ahead.

ALL THE FEATURES

Clapped has most of the features that any premium modbot has. Autorole? yep. dice roller? absolutely. Even YT audio playback!

Have a new idea? send me a message and i'll add it!
